# Primitive DB

Учебный проект: примитивная база данных с управлением таблицами через CLI.

Проект реализован на Python и распространяется как пакет.

## Управление таблицами

После установки пакета в терминале доступна команда запуска базы данных.

### Доступные команды

- `create_table <имя> <столбец:тип> ...` — создать таблицу  
- `list_tables` — показать список всех таблиц  
- `drop_table <имя>` — удалить таблицу  
- `help` — показать справку  
- `exit` — выход из программы  

### Пример использования

```text
$ database

***База данных***
Введите команду (help — справка)

Введите команду: create_table users name:str age:int is_active:bool
Таблица "users" успешно создана со столбцами: ID:int, name:str, age:int, is_active:bool

Введите команду: list_tables
- users

Введите команду: drop_table users
Таблица "users" успешно удалена.

Введите команду: exit
До свидания!
